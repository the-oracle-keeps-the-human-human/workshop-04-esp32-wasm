WASM on ESP32 — Web Flasher

Flash a live WebAssembly device straight from this browser tab — no toolchain, no esptool. Desktop Chrome / Edge / Opera (Web Serial).

This workshop runs WebAssembly on a microcontroller: one tiny .wasm in the browser, on the desktop, and on the chip (wasm3 + WAMR). Flash the desk-pet, or flash a classmate's submission firmware below.

Desk-pet · choose a character

Plug in the native USB port → click → pick the usbmodem port → Install. App image at 0x10000 + shared 3 MB LittleFS at 0x290000 (all packs live there; only the boot default differs). JC3248W535 · ESP32-S3.

This browser can't do Web Serial. Use desktop Chrome, Edge, or Opera.
