WASM on ESP32 — Web Flasher

Flash a live WebAssembly desk-pet to a Guition JC3248W535 (ESP32-S3) straight from this browser tab — no toolchain, no esptool.

This workshop is about running WebAssembly on a microcontroller: one tiny .wasm core that runs in the browser, on the desktop, and on the chip itself (wasm3 + WAMR). The desk-pet below decodes its animations with that same on-chip WASM pipeline — pick a character and flash it to see it run. Then read the guide and submit your own.

🦁 Singhasingha — wasm3 sentinel

A 106-byte zero-import .wasm run on the ESP32 via wasm3. Exports roar(n)=1+…+n and guard(a,b)=a²+b²; after flash, serial @115200 prints roar(10)=55 PASS · guard(3,4)=25 PASS.

Built for plain ESP32 (esp32dev). Submission: 03-singhasingha. 🦁 ฟ้าร้องก่อนฝน — สิงห์เฝ้าโค้ดก่อน production
