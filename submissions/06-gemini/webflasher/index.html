Gemini Oracle ESP32

Flash the Gemini Oracle WASM firmware. The firmware embeds a zero-import WebAssembly module and runs it on the ESP32 with wasm3.

wasmgemini_sum(12,25) = 37
pack sizegemini_pack_size() = 37
skill checkgemini_skill_check(6) = 36
Use Chrome or Edge on desktop. Web Serial is required. Serve this page from HTTPS or localhost before flashing.

Manifest: manifest-gemini-oracle.json. Firmware binaries are staged from the PlatformIO build by ./stage.sh.
